Brick Breaker Game (Java)
📌 Project Overview

Brick Breaker is a simple console-based Java game inspired by the classic arcade Breakout game.
The objective of the game is to destroy all the bricks using a bouncing ball before running out of lives. ()

This project demonstrates Object-Oriented Programming (OOP) concepts such as classes, encapsulation, and modular design.

🎮 Game Description

The game contains a board with bricks placed in the upper rows.
A ball moves upward and hits the bricks based on the player's command.
When the ball hits a brick, the brick gets destroyed and the player loses one life.

The game continues until:

All lives are lost

The player exits the game

⚙️ Technologies Used

Java

Object-Oriented Programming

Eclipse IDE

Command Line Interface
